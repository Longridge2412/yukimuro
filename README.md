# ❄️ 雪室出入庫管理

雪室の入庫・出庫をタブレットで記録し、Googleスプレッドシートで自動管理するWebアプリです。

## 概要

- **作業者**（米農家）がタブレットで入庫・出庫を記録
- **管理者**がGoogleスプレッドシートで在庫確認・請求管理
- 記録は自動でスプレッドシートに反映されます
- パスワード認証でアクセスを制限

## ファイル構成

```
index.html              … 作業者用Webアプリ（タブレットで使用）
gas/Code.gs             … Google Apps Script（スプレッドシート連携用）
雪室管理シート.xlsx      … Excelテンプレート（参考用）
```

## セットアップ手順

### 1. Googleスプレッドシートを作成

1. [Google スプレッドシート](https://sheets.google.com) で新規スプレッドシートを作成
2. メニューの **拡張機能 → Apps Script** を開く
3. `gas/Code.gs` の内容をすべてコピーして貼り付け
4. **保存**（Ctrl+S）
5. `setupSpreadsheet` 関数を実行（▶ ボタン）→ シートが自動作成されます
6. 初回は権限の承認を求められるので「許可」してください

### 2. APIキーを設定（任意）

不正アクセスを防ぐため、APIキーの設定を推奨します:

1. `Code.gs` の `API_KEY` 変数にキーを入力（例: `const API_KEY = "mySecretKey123";`）
2. アプリの管理者設定でも同じキーを入力します（後述）

### 3. マスタデータを入力

スプレッドシートの各シートにデータを入力します:

- **利用者マスタ**: 作業者の名前を入力（No. と 利用者名 が必須）
- **雪室一覧**: 雪室の名前を入力（No. と 雪室名 が必須）
- **品目マスタ**: 米の品種や日本酒の銘柄を入力（プリセット9品種が入っています）

### 4. GASをWebアプリとしてデプロイ

1. Apps Script エディタで **デプロイ → 新しいデプロイ**
2. 種類: **ウェブアプリ**
3. 設定:
   - 説明: 「雪室出入庫管理API」
   - 次のユーザーとして実行: **自分**
   - アクセスできるユーザー: **全員**
4. **デプロイ** をクリック
5. 表示される **Web アプリの URL** をコピー

### 5. アプリを公開

GitHub Pages などで `index.html` をホスティングするか、タブレットのブラウザで直接開きます。

### 6. アプリにGAS URLを設定

1. アプリをブラウザで開く
2. パスワード `yukimuro2026` を入力してログイン
3. ヘッダー部分を **3秒長押し** → 管理者モードを表示
4. **⚙️ 設定 → 🔗 GAS接続設定** を開く
5. GAS WebアプリURLを貼り付け
6. APIキーを設定した場合はそちらも入力
7. **接続テスト** で動作確認 → **保存して反映**

## 使い方

### 作業者（タブレット）

1. ブラウザでアプリを開く
2. パスワードを入力してログイン
3. **入庫** または **出庫** をタップ
4. 画面の案内に従って: 作業者 → 雪室 → 品目 → 数量 を選択
5. 確認画面で **記録する** をタップ
6. スプレッドシートに自動で書き込まれます

### 管理者

- **スプレッドシート**: 入出庫記録・在庫サマリー・請求レポートをリアルタイムで確認
- **アプリの管理者モード**: ヘッダー部分を3秒長押しで表示。在庫・履歴の確認やGAS接続設定ができます

## スプレッドシート構成（6シート）

| シート名 | 内容 |
|---------|------|
| 利用者マスタ | 作業者の名前・連絡先（→アプリの選択肢に反映） |
| 雪室一覧 | 雪室名・場所（→アプリの選択肢に反映） |
| 品目マスタ | 米の品種・日本酒の銘柄（→アプリの選択肢に反映） |
| 入出庫記録 | アプリから自動書き込み |
| 在庫サマリー | 手動で数式を設定して集計（テンプレート参照） |
| 請求レポート | 保管料単価を入力して請求額を算出 |

## オフライン対応

GAS未接続時（ネットワーク障害など）でもローカルに記録が保存されます。接続復帰後に管理者モードからCSV/JSONでエクスポートできます。

## セキュリティ

- **パスワード認証**: アプリ起動時にパスワード入力が必要
- **APIキー**: GAS側で設定可能。アプリとGAS間の通信を保護
- **GAS URL非公開**: URLはコードに含まれず、管理者が端末ごとに設定（localStorageに保存）

## 技術スタック

- HTML / React 18 / Tailwind CSS（単一ファイル）
- Google Apps Script（Webアプリとしてデプロイ）
- Google スプレッドシート（データベース代わり）
